Portugol - Introdução à Lógica de Programação

Portugol é uma forma de pseudocódigo utilizada para ensinar lógica de programação. Ele utiliza comandos em português, facilitando o entendimento para iniciantes que ainda não conhecem linguagens como Java, Python ou C.

O principal objetivo do Portugol é ensinar como pensar de forma lógica, criando algoritmos para resolver problemas passo a passo.

O que é um Algoritmo?

Um algoritmo é uma sequência de passos organizados para resolver um problema. Por exemplo, uma receita de bolo é um algoritmo, pois segue uma ordem lógica de instruções.

Estrutura Básica de um Programa em Portugol

Algoritmo "NomeDoPrograma"
Var
    variavel: tipo
Inicio
    // comandos
FimAlgoritmo

Todo programa em Portugol possui:

  • Algoritmo – nome do programa
  • Var – declaração de variáveis
  • Inicio – começo das instruções
  • FimAlgoritmo – final do programa

Exemplo 1: Escrevendo uma Mensagem

Algoritmo "OlaMundo"
Inicio
    Escreva("Olá, mundo!")
FimAlgoritmo

O comando Escreva serve para mostrar mensagens na tela.

Exemplo 2: Usando Variáveis

Algoritmo "Soma"
Var
    num1, num2, resultado: inteiro
Inicio
    Leia(num1)
    Leia(num2)
    resultado <- num1 + num2
    Escreva("Resultado: ", resultado)
FimAlgoritmo

O comando Leia serve para receber dados do usuário.

Estrutura Condicional (Se...Senao)

A estrutura condicional permite que o programa tome decisões. 

Algoritmo "MaiorIdade"
Var
    idade: inteiro
Inicio
    Leia(idade)
    Se idade >= 18 Entao
        Escreva("Maior de idade")
    Senao
        Escreva("Menor de idade")
    FimSe
FimAlgoritmo

Estrutura de Repetição (Enquanto)

A estrutura de repetição permite executar comandos várias vezes. 

Algoritmo "Contador"
Var
    contador: inteiro
Inicio
    contador <- 1
    Enquanto contador <= 5 Faca
        Escreva(contador)
        contador <- contador + 1
    FimEnquanto
FimAlgoritmo

Nesse exemplo, o programa mostra os números de 1 até 5.
